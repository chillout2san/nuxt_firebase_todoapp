# Ta-Su-Ku

### 概要

Ta-Su-Kuとは直感的に利用することができるタスク管理アプリです。

### 使用した技術やフレームワーク等

1.Nuxt.js
2.TypeScript
3.Firebase(Authentication,Cloud Firestore)
4.Bulma
